## 🧩 How it works

S3T-Former uses a spike-driven state-space topology transformer. In simple terms, it reads motion patterns in skeleton data and looks for action shapes over time. It is meant for tasks like:

- Human action recognition
- Motion classification
- Skeleton-based analysis
- Test runs on action datasets
